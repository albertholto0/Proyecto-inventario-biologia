# Sistema de Gestión de Inventario para Laboratorio de Biología

![Badge en Desarrollo](https://img.shields.io/badge/Estado-En%20Desarrollo-yellow)
![Licencia](https://img.shields.io/badge/Licencia-MIT-blue)

Aplicación web para la gestión del inventario del Laboratorio de Biología de la Universidad de la Sierra Juárez.

## Descripción del Proyecto

Este sistema fue desarrollado como parte del curso de Aplicaciones Web en la Universidad de la Sierra Juárez. Su objetivo principal es digitalizar y optimizar la gestión de los recursos materiales del laboratorio, facilitando:

- Registro centralizado de materiales y equipos
- Control de existencias y caducidad de reactivos
- Seguimiento de préstamos y uso de materiales
- Generación de reportes automáticos

## Características Principales

✅ Interfaz intuitiva y responsive  
✅ Sistema de autenticación para diferentes roles (admin, profesores, estudiantes)  
✅ Búsqueda y filtrado avanzado de materiales  
✅ Notificaciones de caducidad de reactivos  
✅ Generación de reportes en PDF/Excel
